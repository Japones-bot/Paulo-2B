O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar conteúdo na web.

Programação

Qual a estrutura básica de um documento HTML?

A estrutura básica de um documento HTML inclui as tags: DOCTYPE html, html, head, title, body, e /body, /html.
